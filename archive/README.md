# Archivio dei file superati

Questa cartella conserva file non più usati dal percorso operativo corrente. Non contiene dati
email e non deve essere usata come sorgente per nuove esecuzioni.

- `windows-launchers/start_gui.bat`: vecchio launcher della GUI Tkinter, sostituito dal menu
  `EMAIL_ATLAS.bat` e dalla console web locale `AVVIA_CONSOLE.bat`.

La documentazione storica è conservata separatamente in `docs/archive/`; i rapporti progettuali
storici sono in `reports/archive/`.
