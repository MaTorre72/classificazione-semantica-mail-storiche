# Archivio della documentazione storica

Questa cartella conserva le versioni precedenti della documentazione, le proposte UX superate e
le guide delle interfacce legacy. I file non sono stati cancellati, così restano disponibili per
tracciabilità e confronto storico.

Non usare queste pagine come istruzioni operative correnti. Il percorso aggiornato parte da:

1. [`README.md`](../README.md) per l'indice;
2. [`guida_rapida.md`](../guida_rapida.md) per l'avvio;
3. [`guida_uso_completa.md`](../guida_uso_completa.md) per il manuale;
4. [`comandi.md`](../comandi.md) per la CLI.

Le pagine archiviate includono:

- versioni numerate `versione_2_*`, `v3_*` e `v3_1_*`;
- vecchie guide GUI, interfaccia e uso operativo;
- proposte UX e revisione umano/LLM;
- roadmap e documenti di ripensamento ormai storici;
- guide legacy su classificazione, Atlante, LLM e aggiornamento periodico.
