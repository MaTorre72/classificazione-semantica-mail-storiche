# Corrispondenza dei termini tecnici

Questa pagina serve a chi ispeziona database, API o comandi avanzati.

| Nome visibile | Nome interno |
|---|---|
| Area | `macro_category` |
| Insieme | `operational_context` |
| Etichetta | `taxonomy_label` |
| Argomento | `technical_domain` |
| Tipo di insieme | `context_type` |
| Cliente / Ente | `client_or_entity` |
| Stato | `review_status` |
| Affidabilità | `confidence` |
| Azione consigliata | `suggested_user_action` |

I nomi interni restano invariati per garantire retrocompatibilità con dati e comandi precedenti.
