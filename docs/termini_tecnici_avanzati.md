# Termini tecnici avanzati

La guida base usa Archivio, Email, Conversazione, Soggetto, Contesto, Tema operativo, Categoria e
Atlante. Internamente FTS5 gestisce la ricerca; Sentence Transformers può produrre vettori locali;
algoritmi di raggruppamento legacy possono suggerire candidati ma non definiscono l'Atlante.

Le tabelle `atlas_*` sono il nuovo modello conversation-first. Le tabelle precedenti restano per
compatibilità e confronto.
