# Fix Pipeline

Obiettivo: correggere un problema reale della pipeline `email-atlas` senza allargare il perimetro.

Istruzioni:

1. riproduci il problema con il comando piu piccolo possibile;
2. individua il punto preciso del codice;
3. applica la modifica minima sicura;
4. aggiungi o aggiorna test;
5. esegui quality checks mirati;
6. aggiorna backlog e run log.

Vincoli:

- niente GUI nuova;
- niente cloud;
- niente dati reali fuori dal workspace;
- niente scorciatoie che nascondono invalidazioni o perdita di stato.
